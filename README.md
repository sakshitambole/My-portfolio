# My-portfolio
 🌐 Personal Portfolio Website

A responsive and modern personal portfolio website developed using **HTML5** and **CSS3**. This project showcases my education, experience, projects, and personal profile in a clean and user-friendly interface.

---

## 📌 Features

- Responsive design
- Sticky navigation bar
- Home section
- Education section
- Experience section
- Projects section
- Clean and simple UI
- Easy navigation using anchor links

## 🛠️ Technologies Used

- HTML5
- CSS3
