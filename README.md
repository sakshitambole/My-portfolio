# My-portfolio
 🌐 Personal Portfolio Website

A responsive and modern personal portfolio website developed using **HTML5** and **CSS3**. This project showcases my education, experience, projects, and personal profile in a clean and user-friendly interface.

---

## 📌 Features

- Responsive design
- Sticky navigation bar
- Home section
- Education section
- Experience section
- Projects section
- Clean and simple UI
- Easy navigation using anchor links

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

Portfolio-Website/
```
Portfolio-Website/
│── index.html
│── style.css
└── README.md
```
## Installation

1. Clone the repository:

```bash
https://github.com/sakshitambole/My-portfolio.git
```

2. Open the project folder.
  ```bash
cd My-portfolio
```

4. Open `index.html` in your web browser.
